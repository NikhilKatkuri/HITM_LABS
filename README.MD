# HITM_LABS

A unified repository containing all laboratory programs from various subjects in HITM curriculum.  
This project is aimed at helping students easily access lab codes, learn concepts, and revise during exams.

---

## Purpose

- Maintain **all lab codes in one place**
- Improve understanding with well-structured examples
- Help students revise lab programs quickly
- Encourage collaboration and contribution
- Keep code + syllabus organized and accessible
